# Z00 Online

Password-protected landing page with occult minimalist aesthetic.

## Features

- Password-protected entry (password: "meow")
- Folder-based gallery system (password: "rawrxd")
- Countdown timer to March 21, 2026 at 22:30 CET
- Eerie breathing animations
- Hidden second folder (no password required)

## Deployment

This project is deployed on Vercel at [z00.online](https://z00.online)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
